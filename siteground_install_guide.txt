BEXO CARGO - SITEGROUND INSTALLATION GUIDE
=========================================

This guide will help you set up Bexo Cargo on your SiteGround hosting account.

STEP 1: DATABASE SETUP
---------------------
1. Log in to your SiteGround Site Tools
2. Go to Site > MySQL/PostgreSQL
3. Create a new PostgreSQL database:
   - Name: bexocargo_db (or your preferred name)
   - User: Create a new user with a strong password
4. Note down the database credentials

STEP 2: UPLOAD APPLICATION FILES
-------------------------------
1. Extract the ZIP file you downloaded
2. Using Site Tools > File Manager or FTP:
   - Create a directory for the application (e.g., "bexocargo" in your public_html folder)
   - Upload all files to this directory

STEP 3: CONFIGURE THE APPLICATION
-------------------------------
1. Edit the .htaccess file:
   - Replace "USERNAME" with your actual SiteGround username

2. Edit the passenger_wsgi.py file:
   - Replace "USERNAME" with your actual SiteGround username
   - Adjust the Python path if necessary

3. Create a .env file with your environment variables:
   - DATABASE_URL=postgresql://username:password@localhost:5432/database_name
   - SESSION_SECRET=your_secret_key
   - ADMIN_USERNAME=your_admin_username
   - ADMIN_PASSWORD=your_admin_password
   - MAIL_SERVER=your_mail_server
   - MAIL_PORT=587
   - MAIL_USERNAME=your_email
   - MAIL_PASSWORD=your_email_password
   - MAIL_DEFAULT_SENDER=noreply@yourdomain.com

STEP 4: SET UP PYTHON ENVIRONMENT
-------------------------------
1. Access your SiteGround SSH Terminal:
   - Go to Site > SSH Access > Terminal
   
2. Navigate to your application directory:
   - cd public_html/bexocargo (or your chosen directory)
   
3. Create a virtual environment:
   - python3 -m venv venv
   
4. Install dependencies:
   - source venv/bin/activate
   - pip install -r requirements_list.txt

STEP 5: INITIALIZE THE DATABASE
-----------------------------
1. Still in the SSH Terminal, with your virtual environment activated:
   - python
   - from app import app, db
   - with app.app_context():
   -     db.create_all()
   - exit()

STEP 6: TEST YOUR APPLICATION
---------------------------
1. Visit your website (https://yourdomain.com)
2. Make sure everything works correctly
3. Test the admin interface (https://yourdomain.com/admin)
   - Default login: admin / admin123 (unless you changed it in the .env file)

TROUBLESHOOTING
--------------
1. Check your SiteGround error logs if anything goes wrong
2. Make sure you've edited the correct paths in passenger_wsgi.py and .htaccess
3. Ensure your virtual environment is properly set up
4. Verify database credentials in your .env file
